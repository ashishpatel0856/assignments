# Hello Names 
Hello Names is a simple full-stack web applications which is implemented by Spring boot and React.js. In this allows users to submit names and then to see a list of all submitted names.

The main goal of this project is to demonstrate basic full-stack development skills, API integration, clean code structions  with documentation.

## Tech Stack used

1->  ### Backend
       -java
       -spring boot
       -spring web
       -validations
       -dependencies


2->  ### Frontend
      -React with vite
      -tailwind css
      -axios

## Featues of Projects
- Add a new name using a simple form
- -display all name after submit button
- -backend validation for name input , must be non-empty name
- -no database uses
- clena and responsive ui

## Prerequisites
- java 21
- maven

- using run button available in intellij
